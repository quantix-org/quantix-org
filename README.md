# Quantix Protocol

Quantix is a post-quantum blockchain protocol built with SPHINCS+ cryptography and PBFT consensus.

## Overview

Quantix provides a secure, scalable blockchain infrastructure with:
- Post-quantum cryptography (SPHINCS+/STHINCS)
- PBFT consensus mechanism
- EVM-compatible smart contracts via SVM
- DHT-based P2P networking

## Getting Started

```bash
go mod tidy
go build ./...
```
